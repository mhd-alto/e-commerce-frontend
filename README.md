# Starry Night Admin Dashboard

A responsive admin dashboard built with **HTML + Bootstrap 5** (CDN) and **Font Awesome**, themed with a custom CSS stylesheet inspired by **Vincent van Gogh’s *The Starry Night***.

## Files
- `index.html` – layout, tables, modals
- `styles/starry.css` – Starry Night theme
- `app.js` – mock stats + modal form population + order status badge updates

## Run
Open `admin-dashboard/index.html` in a browser.

> Note: This is a static UI mock. Data is mock and actions are client-side only.
